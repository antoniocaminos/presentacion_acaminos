JavaScript


JavaScript es un lenguaje de programación que se usa para agregar interactividad y dinamismo a las páginas web.
A diferencia de HTML y CSS, que definen la estructura y el estilo visual de una página, JavaScript permite que los
elementos de una página web respondan a las acciones de los usuarios (como hacer clic, mover el ratón o escribir texto)


Conceptos Básicos de JavaScript

Variables:Crea una variable cuyo valor no puede cambiar (constante)


  1. let:Crea una variable cuyo valor puede cambiar.
  2. const:Crea una variable cuyo valor no puede cambiar (constante)
  3. var:(poco usado hoy en día) una forma anterior de declarar variables, con algunas diferencias en su comportamiento.


Funciones: Las funciones son bloques de código que pueden ejecutarse cuando se llaman. Permiten agrupar instrucciones para realizar tareas específicas.






Eventos:

Los eventos en JavaScript son acciones que ocurren en la página, como hacer clic, pasar el ratón sobre un elemento o presionar una tecla.
Podemos usar JavaScript para responder a estos eventos.


DOM (Document Object Model):

El DOM es una representación estructurada de una página web que permite a JavaScript
interactuar con el HTML y cambiar contenido, estilos y atributos en tiempo real.
