CSS

CSS (Cascading Style Sheets) es el lenguaje usado para diseñar y personalizar el estilo de las páginas web. Permite cambiar el color, la tipografía, el diseño y la distribución de los elementos en HTML.
Con CSS, transformamos una estructura HTML básica en algo visualmente atractivo y profesional.


    Tres métodos para incluir CSS en una página

  1. CSS en línea: se usa en etiquetas específicas dentro del HTML usando el atributo style
  2. CSS interno: se agrega en la misma página, dentro de una etiqueta <style> en el <head>
  3. CSS externo: se enlaza un archivo CSS externo, permitiendo modularizar estilos para múltiples páginas.

fuente: https://www.w3schools.com/

Ejemplo de estilos en CSS

h1{
color: #3366cc; /* Cambia el color del texto */
font-size: 2.5em; /* Cambia el tamaño del texto */
text-align: center; /* Alinea el texto al centro */
}


p{ color: #333333;
font-family: Arial, sans-serif; /* Define la tipografía */
line-height: 1.6; /* Define el espaciado entre líneas */
text-align: justify;
}


.navbar {
background-color: #f8f9fa; /* Fondo para la barra de navegación */
padding: 10px;
border-radius: 5px; /* Bordes redondeados */
}


.button {
background-color: #4CAF50; /* Verde */
color: white;
padding: 10px 20px;
text-align: center;
text-decoration: none;
display: inline-block;
font-size: 16px;
margin: 4px 2px;
cursor: pointer;
}
.button:hover {
background-color: #45a049; /* Cambia el color al pasar el ratón */
}
