Introducción a SQL

SQL (Structured Query Language) es un lenguaje diseñado para gestionar y manipular bases de datos. Permite crear, leer, actualizar y eliminar datos en sistemas de bases de datos relacionales.

Crear Tablas

Para crear una tabla en SQL, usamos el comando CREATE TABLE seguido del nombre de la tabla y la definición de sus columnas.

        <code>
        CREATE TABLE Usuarios (
            ID INT PRIMARY KEY,
            Nombre VARCHAR(50),
            Edad INT,
            Email VARCHAR(100)
        );
        </code>

Insertar Datos

Para insertar datos en una tabla, usamos INSERT INTO seguido del nombre de la tabla y los valores a agregar.

        <code>
        INSERT INTO Usuarios (ID, Nombre, Edad, Email) 
        VALUES (1, 'Juan Pérez', 28, 'juan.perez@example.com');
        </code>

Consultar Datos

Para obtener datos de una tabla, usamos SELECT junto con el nombre de las columnas o * para seleccionar todas.

        <code>
        SELECT * FROM Usuarios;
        </code>

También podemos aplicar condiciones con WHERE para filtrar los resultados.

        <code>
        SELECT Nombre, Email FROM Usuarios WHERE Edad > 18;
        </code>

Actualizar y Eliminar Datos

Para actualizar datos, usamos UPDATE seguido de SET y luego WHERE para especificar los registros.

        <code>
        UPDATE Usuarios 
        SET Email = 'nuevo.email@example.com' 
        WHERE ID = 1;
        </code>

Para eliminar registros, usamos DELETE FROM seguido de WHERE.

        <code>
        DELETE FROM Usuarios WHERE ID = 1;
        </code>
