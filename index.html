Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado usado para estructurar el contenido en la web. Usamos etiquetas para definir diferentes tipos de contenido y estructura en una página.

Ejemplos de etiquetas básicas:

  • <html>: Define el documento como HTML.
  • <head>: Contiene información de configuración, como el título y enlaces a CSS o JavaScript.
  • <title>: Define el título de la página que aparece en la pestaña del navegador.
  • <body>: Contiene el contenido principal de la página, como texto, imágenes, videos, etc.
  • <h1> a <h6>: Son encabezados con diferentes niveles de importancia, siendo <h1> el más importante.
  • <p>: Define un párrafo de texto.

Otras etiquetas:


<br>: Salto de línea (no tiene etiqueta de cierre).
<strong>: Aplica énfasis en negrita (usualmente se muestra en negrita).
<em>: Aplica énfasis en cursiva (usualmente se muestra en cursiva).
<blockquote>: Cita en bloque, para citas largas (con indentación).
Listas:
<ul>: Lista desordenada (bullets).
<ol>: Lista ordenada (números).
<li>: Elemento de lista (usado dentro de <ul> o <ol>).
Imágenes y multimedia:
<img src="ruta-imagen.jpg" alt="Descripción de la imagen">: Inserta una imagen en la página. La etiqueta alt describe la imagen para accesibilidad.
<audio controls src="ruta-audio.mp3"></audio>: Inserta un reproductor de audio con controles.
<video controls width="320" height="240" src="ruta-video.mp4"></video>: Inserta un reproductor de video con controles.
Enlaces:
<a href="https://www.ejemplo.com">Texto del enlace</a>: Crea un enlace a otra página o recurso.
<a href="#seccion">Enlace a una sección de la misma página</a>: Crea un enlace a una sección específica dentro de la misma página usando un id.
Tablas:
<table>: Define una tabla.
<tr>: Define una fila de la tabla.
<th>: Encabezado de la tabla (suele estar en la primera fila).
<td>: Celda de datos de la tabla.
<caption>: Título de la tabla.
Formularios:
<form>: Define un formulario.
<input type="text" name="nombre" placeholder="Escribe tu nombre">: Campo de texto para introducir datos.
<button type="submit">Enviar</button>: Botón para enviar el formulario.
<label for="email">Correo:</label><input type="email" id="email" name="email">: Crea un campo de entrada de correo electrónico con un label asociado.
Estructura y agrupación:
<div>: División de contenido. Se usa como contenedor para agrupar elementos.
<span>: Contenedor en línea, útil para aplicar estilos a partes específicas de texto.
<header>: Define la cabecera de una página o sección.
<footer>: Define el pie de página.
<section>: Define una sección de contenido relacionada.
<article>: Define un contenido independiente (como un artículo o post de blog).
<aside>: Define contenido adicional (como una barra lateral).
Etiquetas Semánticas (HTML5):
<main>: Contenido principal de la página.
<nav>: Define una sección de navegación, generalmente para enlaces.
<figure>: Contenedor para imágenes con leyenda.
<figcaption>: Leyenda para la imagen dentro de <figure>.
